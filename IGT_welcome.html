🎰アイオワギャンブリング課題

ゲームルール説明

📝 課題の目的

この課題は、意思決定能力と学習能力を測定します。あなたは4つの札の山から繰り返し選択を行い、利益を最大化することを目指します。

🎮 ゲームの流れ

課題は4つのブロックで構成され、各ブロックで16回の試行を行います。（合計64試行）

  • 初期資金: 2,000
  • 各ブロック: 16回の選択
  • ブロック間: 資金がリセットされます
  • 最終結果: すべてのブロックの成績を集計します

🃏 4つの札の山

4つの異なる札の山から1つを選びます。各山の特性は以下の通りです：

山 A
高リスク・高リターン
💰 獲得: +100
💔 損失: −250
山 B
超高リスク・高リターン
💰 獲得: +100
💔 損失: −1,250
山 C
低リスク・低リターン
💰 獲得: +50
💔 損失: −50（50%）
山 D
低リスク・低リターン
💰 獲得: +50
💔 損失: −200（50%）

⚠️ 重要: 山 C と D は 50%の確率で損失がありません。これらは長期的に有利ですが、山 A・B よりもリターンが小さいです。

📊 試行の流れ

各試行では以下の手順を繰り返します：

  1. 4つの札の山（A, B, C, D）から1つを選択します
  2. その山から札を1枚引きます
  3. 獲得額と損失額が表示されます
  4. 「次へ」ボタンをクリックして次の試行に進みます
例）山 C を選んだ場合：
獲得: +50 / 損失: −0 → +50 の利益

例）山 A を選んだ場合：
獲得: +100 / 損失: −250 → −150 の損失

💡 ポイント

  • 学習が重要: 最初は各山の特性が分かりませんが、試行を重ねるにつれて学習します
  • リアルタイム表示: 現在の資金と試行数はいつでも確認できます
  • ペース管理: 自分のペースで進めてください。時間制限はありません
  • 戦略性: 短期的な利益と長期的な最適性のバランスが求められます

📈 結果表示

各ブロック終了時と課題全体終了時に以下の統計が表示されます：

  • 最終資金
  • 総収支（利益・損失）
  • 各山の選択回数
  • 有利な山（C・D）の選択率
